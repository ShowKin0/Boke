⚙️ 后台管理 — 架构方案

有了后台管理需求，需要引入服务端，以下是两种方案

🟢 方案 A：Node.js + Express

推荐

  • Express 服务端 + JSON 文件存储
  • 静态文件托管前端页面
  • RESTful API 提供 CRUD 操作
  • 简单密码认证
  • 管理员后台独立页面 /admin
  • 部署简单，无需数据库

🟡 方案 B：纯静态 + localStorage

简化版

  • 无需服务端，纯前端
  • 数据存 localStorage
  • 管理员模式在页面内切换
  • 数据仅当前浏览器可见
  • 无法多设备同步

📋 后台管理功能

📝 文章管理
新增/编辑/删除/置顶
💬 动态管理
发布/编辑/删除动态
🔍 探索管理
新增/编辑/分类/删除链接
🎵 音乐管理
添加/删除歌曲、外链歌单
🎨 主题配色
自定义主色、粉/蓝/背景色等 CSS 变量，实时预览
